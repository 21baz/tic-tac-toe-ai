## Tic-Tac-Toe (Web Game)

A Tic-Tac-Toe game with:
- Play vs AI (multiple difficulties)
- Custom player colours
- Win detection + confetti
- Responsive UI


## Planned Improvements

While the game is fully playable, there are several enhancements I’d like to implement in future iterations:

- **Alternating starting player**  
  Currently, Player X always goes first. A fairer system would alternate which player starts each round, or randomly select the starting player.

- **Smarter AI decision-making**  
  While the AI supports multiple difficulty levels, its strategy could be improved further with deeper lookahead or optimisation for edge cases.

- **Mobile-first responsiveness**  
  Improve spacing, button sizing, and animations specifically for smaller screens and touch interaction.

- **Sound & animation controls**  
  Allow users to toggle sound effects and animations on/off for preference or accessibility.

- **Persistent score tracking**  
  Store scores using localStorage so results persist across page refreshes.

- **Game statistics**  
  Track wins, losses, and draws over time to give players insight / analytics into their performance.

- **Multiplayer enhancements**  
  Add an online multiplayer mode using WebSockets or a simple backend service.
